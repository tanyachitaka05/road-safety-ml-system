Machine Learning 

 Instructions for Running the Notebook
- Open Jupyter Notebook or VS Code
- Ensure all dependencies are installed
- Run cells in order from top to bottom
- Ensure datasets are placed in the correct directory

Python Version Used
- Python 3.10 or higher recommended

 List of Dependencies
- pandas
- numpy
- scikit-learn
- xgboost
- catboost
- matplotlib
- seaborn
- torch


 Expected Runtime
- Data preprocessing: 2–5 minutes
- Model training: 10–40 minutes depending on hardware


 Hardware Requirements
- Minimum: 8GB RAM, CPU
- Recommended: GPU
